韵匣 RimeKit v0.1.0 首次公开发布。本项目通过 vibe coding 开发。

这是韵匣的第一个公开发布版本。当前版本仅包含 Windows 端，Android 端仍在开发中。

包含的功能：
- 小狼毫输入法的一键安装、卸载和完全清理
- 薄荷拼音输入方案的安装和卸载
- 词库（moetype ACG词库、搜狗网络流行新词、zhwiki百科词库）的安装和卸载
- 万象语法模型的安装和卸载
- 候选窗全部外观设置（字体、字号、配色、布局、阴影、圆角等）
- 输入行为设置（简繁、全半角、中英文标点、Emoji、声调、模糊音）
- 自定义词条管理（新增、删除、导入）
- 命令行工具（22 个命令），支持自动化脚本和 AI agent 调用

系统要求：
- Windows 10 或更高版本，64位
- 无需额外安装运行环境

使用方法：
1. 下载 RimeKit.Windows.Gui.exe，双击运行
2. 首次使用建议顺序：安装小狼毫 → 安装薄荷方案 → 安装词库 → 调整设置
3. 命令行版本：在终端运行 RimeKit.Windows.Cli.exe doctor

已知限制：
- 尚未支持 Android 端
- 尚未支持跨端同步
- 部分词库的下载依赖 GitHub API 网络连接
- 中文百科词库（zhwiki）需 GitHub Releases API 可正常访问
